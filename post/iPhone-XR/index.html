iPhone XR 入手一周的喜欢和不喜欢

· 2019-05-01 · # 生活 # 数码

在入手 iPhone XR 128g 红色版一周之后，我写下了这篇算不上评测的文章，来分享一下我对这部手机的喜欢和不喜欢

喜欢

手机手感

iPhone XR 尺寸是150.9mm * 75.7mm * 8.3mm，重量高达 194g，就厚度和重量而言，iPhone XR 在旗舰机中当属最后一档的水平。iPhone XR 是 Apple 全系列在售中除了 iPhone XSMax 以外最沉的，但是上手之后却发现刚好适合我。

手机屏幕

到了大家都在吐槽的一个环节，6.1 寸的 LCD屏幕，1792 x 828 像素分辨率，326 ppi。实话实话，在入手之前很多人都在吐槽这块屏幕的大果粒，然而入手之后却发现这块屏幕很讨喜，虽然没有 OLED 屏幕的那种高级感，但是胜在更适合人眼，起码晚上玩手机不会感到不适，也避免了烧屏的问题。

Face ID

在使用Face ID之前，我一直都是在使用指纹解锁的形式。指纹好处多种多样，不受环境的限制，解锁速度快等等，但是背后开洞的方式实在是有缺美观。Face ID 成功地解决了背后开洞问题，但是也引起了其他问题，就我上手这段时间看来，Face ID 在黑暗环境中的表现一般，摘下眼镜解锁失败率也很高，尽管如我，我还是更偏爱 Face ID。

手机续航

iPhone XR 的大电池基本可以维持我一天的使用，确实是电池表现最好的一代 iPhone。

不喜欢

手机大边框

实话实话，在刚拿到手的那一刻，iPhone XR 的大边框让我产生了退货的冲动，但是在开机使用一段时间之后，却也适应了很多，如果能把边框做小，这部手机就几乎没有短板了。

手机单摄

在 2018 年底的 DXO 测试中，DXOMark为iPhone XR打出了101分的超高分，排名手机榜第7位，具体得分方面iPhone XR在拍照上获得了103分的高分，录像上获得了96分。尽管如此，我还是觉得要是能上双摄就更好了，哈哈📱📷。

总结

这部 iPhone XR 大概率会陪伴我很长一段时间，真香机绝对是目前市面上最值得购买的 iOS 系列手机。
